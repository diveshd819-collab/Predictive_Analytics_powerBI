# Predictive Analytics Dashboard using Power BI

## Project Overview

This project analyzes historical sales data to identify trends, product performance, and category-wise revenue patterns using Power BI.

## Tools Used

* Power BI
* Excel Dataset

## Features

* Total Revenue Analysis
* Total Orders and Quantity Tracking
* Revenue Trend Visualization
* Revenue by Category Analysis
* Revenue by Product Analysis
* Interactive Filtering using Slicers

## Key Insights

* Identified top-performing products and categories.
* Analyzed historical revenue trends.
* Created interactive dashboards for business decision-making.
* Visualized sales performance using charts and KPIs.

## Outcome

Built an interactive analytics dashboard that transforms raw sales data into meaningful business insights for trend analysis and forecasting support.
